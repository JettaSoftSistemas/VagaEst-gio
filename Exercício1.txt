C++ é uma linguagem de programação de médio nível, pois combina características de linguagens de baixo e alto nível. Foi desenvolvida em 1983 e possui as seguintes características: Suporta múltiplos paradigmas, fornece ao programador escolhas, mesmo que seja possível ao programador escolher a opção errada, é uma linguagem tipada estaticamente e de propósito geral que é tão eficiente e portátil quanto o C, etc.
É uma linguagem bastante utilizada em academias e na criação de diversos aplicativos. As vantagens do C++ são: produção de código eficiente, adequado para grandes projetos, compatível com a linguagem C, padronizado pela ISO, etc. Já as desvantagens são: a compatibilidade com o C causou problemas de entendimento de sintaxe para o C++, grande período de aprendizado, etc.
